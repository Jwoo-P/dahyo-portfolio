## 프로젝트 개요

리스크제로의 산업현장 사고 데이터 분석 기반 위험도 예측 서비스를 해양수산부 현장에 맞게 확장한 안전관리 시스템 기획 프로젝트입니다. 스마트안전장비 데이터 연동을 통해 실시간 통합관제 서비스를 제공하고, 현장 관리자가 현장을 안전하게 관리할 수 있는 관제 솔루션을 도입하는 것이 목표였습니다.

![해양수산부 안전관리 시스템 — 프로젝트 목적](/images/how/maritime-safety/01-purpose.jpg)

## 문제 정의

해양수산부 건설 현장에서는 근로자 출역 관리, 안전 상태 평가, 위험성 평가, CCTV 모니터링이 필요했습니다. 여기에 스마트 안전모·안전밴드, 중장비 기울기 센서, 스마트 아크차단기 등 IoT 장비 데이터를 연동하여 실시간으로 현장 상황을 파악할 수 있는 통합 관제 시스템이 요구되었습니다.

## 맡은 역할

- SI 기획
- 요구사항 분석 및 API 문서 검토
- 모니터링 화면 설계

## 진행 과정

### 요구사항 분석

기본 솔루션 기능으로 출역 관리, 안전 상태 평가, 위험성 평가관리, CCTV 모니터링을 확인했습니다. 스마트 안전 장비 연동 요구사항으로 안전모·밴드 착용자의 위치·신체정보, 중장비 기울기 센서의 위치·기울기 정보, 스마트 아크차단기 상태 정보 제공이 포함되었습니다.

![요구사항 분석 및 개발 방향 선정](/images/how/maritime-safety/02-requirements.jpg)

### API 문서 분석

장비별 신호 값을 확인하고 필요에 따라 분류했습니다. 사용자가 필요로 하는 정보 값, 날짜·정수·텍스트 등 취급하는 정보 형태를 파악하여 화면에 표시할 데이터를 정의했습니다.

### 개발 방향 선정

중장비 상태 모니터링(구글맵 기반 위치·상태 표시, 운전자 정보 매칭, 전복 여부 알림), 스마트 안전모·밴드 착용자 모니터링(출역 근로자 매칭, 체온·심박수·SOS 신호 표시), 스마트 아크차단기 상태 모니터링(현재 상태·정상 유지 기간 표시) 방향으로 기획안을 도출했습니다.

### 화면 설계

시스템 전체 설계와 함께 중장비 상태 표시, 스마트 밴드 상태 표시, 아크 누전 차단기 상태 표시 등 세부 UI 컴포넌트를 추가로 설계했습니다.

![시스템 설계 및 추가 디자인 작업](/images/how/maritime-safety/03-design.jpg)

최종적으로 메인 대시보드, 스마트 안전모·밴드 모니터링 화면, 중장비 모니터링·상세 화면을 완성했습니다.

![메인 대시보드 및 모니터링 화면](/images/how/maritime-safety/04-screenshots.jpg)

## 사용한 도구/방법

- 요구사항 분석 및 기존 솔루션 벤치마킹
- API 문서 분석
- 모니터링 대시보드·상태 컴포넌트 화면 설계

## 협업 방식

리스크제로 SI 프로젝트로 개발팀·디자인팀과 협업했습니다. API 문서를 분석하여 장비별 데이터를 화면에 매핑하고, 현장 관리자가 한눈에 파악할 수 있는 관제 화면을 기획했습니다.

## 결과 및 회고

해양수산부 현장의 출퇴근·안전 점수·센서·CCTV를 통합 관리하는 안전관리 시스템 기획을 완료했습니다. IoT 장비 연동 기반의 실시간 모니터링 화면을 설계하며, SI 기획에서 API 분석과 데이터 시각화 기획의 중요성을 경험할 수 있었습니다.
